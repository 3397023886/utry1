## 快速开始

### 安装和运行

1. 克隆或下载项目
2. 打开终端，进入项目目录
3. 运行以下命令：

```bash
npm install
npm run dev
```

4. 在浏览器中打开 `http://localhost:5173`

## 创建简历

### 方法一：从首页创建

1. 进入应用首页
2. 点击"开始创建"按钮
3. 选择你喜欢的模板：
   - **现代风格** - 简洁现代的设计
   - **经典风格** - 传统专业的设计
   - **创意风格** - 富有创意的设计
4. 点击"使用此模板"

### 方法二：从仪表板创建

1. 点击首页的"📊 我的简历"或访问 `/dashboard`
2. 点击"+ 创建新简历"按钮
3. 选择模板

## 编辑简历

### 个人信息

在"个人信息"标签页中填写：

- **姓名** - 你的全名
- **邮箱** - 联系邮箱
- **电话** - 联系电话
- **位置** - 所在城市或国家
- **职业总结** - 简短的职业介绍或目标

### 工作经验

在"工作经验"标签页中：

1. 点击"+ 添加工作经验"添加新的工作记录
2. 填写以下信息：
   - **公司** - 公司名称
   - **职位** - 你的职位
   - **开始日期** - 工作开始时间
   - **结束日期** - 工作结束时间（如果仍在职，填"至今"）
   - **描述** - 工作职责和成就

### 教育背景

在"教育背景"标签页中：

1. 点击"+ 添加教育背景"添加新的学历
2. 填写以下信息：
   - **学校** - 学校名称
   - **学位** - 学位类型（如学士、硕士等）
   - **专业** - 专业名称
   - **毕业日期** - 毕业时间

### 技能

在"技能"标签页中：

1. 在文本框中列出你的技能
2. 使用逗号分隔多个技能
3. 例如：`JavaScript, React, Node.js, Python`

### 项目

在"项目"标签页中：

1. 点击"+ 添加项目"添加新项目
2. 填写以下信息：
   - **项目名称** - 项目的名称
   - **描述** - 项目的简短描述
   - **链接** - 项目的URL（可选）

## 预览和导出

### 实时预览

- 编辑器右侧会实时显示简历预览
- 当你修改任何信息时，预览会立即更新
- 预览采用专业的排版和格式

### 导出PDF

1. 在编辑页面顶部点击"📥 导出PDF"按钮
2. 浏览器会打开打印对话框
3. 选择"保存为PDF"或"打印到PDF"
4. 选择保存位置并确认

### 打印

1. 点击"导出PDF"按钮
2. 选择"打印"选项
3. 配置打印设置并打印

## 管理简历

### 查看所有简历

1. 在首页或仪表板中查看"我的简历"列表
2. 每个简历显示：
   - 简历标题
   - 使用的模板
   - 最后更新时间

### 编辑简历

1. 点击简历卡片上的"编辑"按钮
2. 进入编辑页面进行修改
3. 修改会自动保存

### 删除简历

1. 点击简历卡片上的"删除"按钮
2. 确认删除（注意：删除后无法恢复）

### 修改简历标题

1. 进入编辑页面
2. 在顶部的标题输入框中修改标题
3. 标题会自动保存

## 常见问题

### Q: 我的数据会保存在哪里？

A: 所有数据都保存在你的浏览器本地存储中。这意味着数据不会上传到任何服务器，完全私密。但如果你清除浏览器缓存，数据可能会丢失。

### Q: 我可以在不同的设备上访问我的简历吗？

A: 目前不行。数据存储在本地，所以只能在同一设备和浏览器上访问。未来版本可能会支持云端同步。

### Q: 如何备份我的简历？

A: 你可以定期导出PDF版本作为备份。未来版本可能会支持导出JSON格式的数据。

### Q: PDF导出的格式是否可以自定义？

A: 目前使用浏览器的打印功能导出PDF。你可以在打印对话框中调整页面设置、边距等。

### Q: 支持哪些浏览器？

A: ResumeStar 支持所有现代浏览器，包括：
- Chrome/Chromium
- Firefox
- Safari
- Edge

### Q: 如何联系支持？

A: 如有问题，请在 GitHub 上提交 Issue 或通过邮件联系我们。

## 技巧和最佳实践

### 1. 定期保存

虽然编辑会自动保存，但建议定期导出PDF作为备份。

### 2. 使用清晰的语言

- 使用具体的动词（如"开发"、"设计"、"管理"）
- 量化成就（如"提高效率30%"）
- 避免冗长的句子

### 3. 保持简洁

- 简历应该清晰易读
- 避免过多的颜色或复杂的格式
- 关键信息应该一目了然

### 4. 针对职位定制

- 根据不同的职位要求调整简历内容
- 强调相关的技能和经验
- 可以创建多个版本的简历

### 5. 定期更新

- 添加新的项目和经验
- 更新技能列表
- 保持简历最新
